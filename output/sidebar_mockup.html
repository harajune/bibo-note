Sidebar Article List with Caching

This mockup demonstrates the sidebar article list functionality with caching.

The sidebar shows the latest articles sorted chronologically by UUIDv7.

The cache is updated when articles are created or modified.

Directory structure:

  • tenant/data/uuid.toml - Article files
  • tenant/cache/article_list_cache.json - Cache files

Implementation Details

The implementation follows these key principles:

  • Repository classes handle all data access operations
  • Each repository implementation determines the appropriate file paths based on environment and tenant
  • Cache files are stored in a dedicated cache directory
  • Article files are stored in a dedicated data directory
  • Multi-tenant support is maintained with proper isolation
